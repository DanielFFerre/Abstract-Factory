Nome: Daniel Felipe Ferrerira

descrição: O Abstract Factory funciona como uma interface de alto nível que agrupa diversas fábricas individuais, mas relacionadas, 
sem especificar suas classes concretas. Em vez de criar um único objeto, ele define um contrato para a criação
de famílias de produtos que devem trabalhar juntos, garantindo a consistência e a compatibilidade entre eles.
